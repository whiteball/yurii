伺かゴースト
「死せる星空ガールズ」

-----

このゴーストは「アジアンパンクTRPG サタスペ」の世界観をベースにしています。
サタスペ世界である犯罪都市オオサカの街でしのぎを削る大犯罪組織たち……の裏でひっそりと、
でも賑やかにハシャぐ亜侠(任侠未満)たちのゴーストです。

「アジアンパンクTRPG サタスペ」は河嶋陶一朗ほか／冒険企画局の著作物です。
http://www.bouken.jp/product/satasupe/

----

登場人物
ユーリィ・ドラコ
ロシア出身の女装男子。
表向きはオオサカでロシアから輸入品の卸を営んでいる。
亜侠としてはその商売関係や私服警官の伝手から依頼を受けて仕事を行っている。
受ける依頼は人や物の捜索、護衛などが多い。趣味の風俗関係から仕事の情報を得ることもある。
ちなみにどっちでもイける。

楊 響子(やん きょうこ)
オオサカ生まれの華人の家出娘。
当然お金がないので、いろいろと「拝借」して生活している。
一応寝る場所は確保しているが、ユーリィの家が快適すぎるので入り浸っている。
ユーリィもはじめは鍵を増やしたりして入らせないようにしていたが、
何をやっても破って入ってくるので、商売道具に手を出さないならと諦められている。
たまに家賃代わりにユーリィの仕事を手伝っている。
